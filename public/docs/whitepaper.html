01. Resumo Executivo

Unlike traditional Proof of Work systems that waste energy, BIGchain rewards nodes for real work: routing and propagating data across the network.

O BIGFOOT Connect é um ecossistema descentralizado construído em torno da BIGchain, uma blockchain leve que recompensa usuários por participarem da rede através de um mecanismo de consenso inovador chamado Proof of Relay (PoR). Ao contrário dos sistemas tradicionais de Proof of Work que desperdiçam energia, a BIGchain recompensa os nós pelo trabalho real que executam: rotear e propagar dados pela rede.

02. Introduction

02. Introdução

Most blockchain networks face a fundamental tension between security, decentralization, and sustainability. Proof of Work systems require enormous energy expenditure. Proof of Stake systems tend to concentrate power among large token holders.

A maioria das redes blockchain enfrenta uma tensão fundamental entre segurança, descentralização e sustentabilidade. Sistemas de Proof of Work exigem enorme gasto energético. Sistemas de Proof of Stake tendem a concentrar poder entre grandes detentores de tokens.

The New Paradigm: Proof of Relay

O Novo Paradigma: Proof of Relay

BIGchain takes a different approach. The question it asks is: what is the real value a network participant offers? The answer is relay—routing requests, propagating blocks, and keeping the network connected.

A BIGchain adota uma abordagem diferente. A pergunta que ela faz é: qual é o valor real que um participante da rede oferece? A resposta é o relay — rotear requisições, propagar blocos e manter a rede conectada.

Why Proof of Relay?
Por que Proof of Relay?

  • Sustainable: Rewards real work, not wasted computation

  • Fair: Anyone can participate with standard hardware

  • Scalable: More nodes make the network stronger

  • Sustentável: Recompensa trabalho real, não computação desperdiçada

  • Justo: Qualquer pessoa pode participar com hardware comum

  • Escalável: Quanto mais nós, mais forte a rede

03. BIGchain - Architecture & Proof of Relay

03. BIGchain - Arquitetura & Proof of Relay

P2P Architecture

Arquitetura P2P

BIGchain is a decentralized peer-to-peer blockchain network where any user can run a node. Nodes communicate via a custom TCP protocol with structured ECDSA handshake, heartbeat system, and cryptographic message signing.

A BIGchain é uma rede blockchain peer-to-peer descentralizada onde qualquer usuário pode rodar um nó. Os nós se comunicam via um protocolo TCP customizado com handshake ECDSA estruturado, sistema de heartbeat e assinatura criptográfica de mensagens.

  • Custom P2P protocol with ECDSA handshake

  • Automatic peer discovery via public API

  • Round-robin leader election for block production

  • Incremental blockchain synchronization

  • Protocolo P2P customizado com handshake ECDSA

  • Descoberta automática de peers via API pública

  • Eleição round-robin de líderes para produção de blocos

  • Sincronização incremental de blockchain

04. BIGFOOT Connect Application

04. Aplicativo BIGFOOT Connect

BIGFOOT Connect is the desktop application that integrates all ecosystem components. Users interact with BIGchain, manage their wallet, monitor relay statistics, and request bridge conversions through a unified interface.

O BIGFOOT Connect é o aplicativo desktop que integra todos os componentes do ecossistema. Usuários interagem com a BIGchain, gerenciam sua carteira, monitoram estatísticas de relay e solicitam conversões de ponte através de uma interface unificada.

05. Token Economics

05. Economia do Token

BIG is emitted through a time-based schedule (not block count). This ensures fairness for users joining later, as daily emission is fixed regardless of how many nodes are active.

O BIG é emitido através de um calendário baseado em tempo (não em contagem de blocos). Isso garante equidade para usuários que entram mais tarde, pois a emissão diária é fixa independentemente de quantos nós estão ativos.
